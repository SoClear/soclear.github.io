# 编程字体推荐

强烈推荐：[Maple Mono](https://github.com/subframe7536/maple-font)

推荐理由：

- 只有它的字体边缘过渡是圆角
- 中英文 2:1 等宽
- 自带的中文也很圆润

推荐下载版本: [Normal-No-Ligature NF-CN unhinted](https://github.com/subframe7536/maple-font/releases/)

- Normal: 正常字体，不会有奇怪样式的 @$%&
- No Ligature: 不会自动连字（合并字符）
- NF: Nerd Fonts 支持
- CN: 包含中文字体
- unhinted: 高分辨率屏幕下效果好
